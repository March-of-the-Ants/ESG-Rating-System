코스닥 상장 중소기업을 대상으로 ESG 평가 지표를 개발하여 개별 기업의 ESG 활동 측정 데이터를 수집하고, <br/>
텍스트 마이닝을 통해 E/S/G 항목별 문서 분류 및 감성 분류 평가 결과를 대시보드 시각화 및 이메일 자동 발송함
